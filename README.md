# BJORN – Custom Voiced Follower with Quests (Scripts)

This repository contains the scripts and supporting files for the mod **BJORN – Custom Voiced Follower with Quests** for *The Elder Scrolls V: Skyrim Special Edition*.

## Mod Overview

**BJORN – Custom Voiced Follower with Quests** is a standalone, lore-friendly follower mod that adds a vanilla plus, lore-friendly, standalone follower. Bjorn has his own quests and backstory, reacts to events and surroundings, interacts with NPCs, and can lead to a deep friendship or a true love story.

For more details, see the mod page on [Nexus Mods](https://www.nexusmods.com/skyrimspecialedition/mods/91652).

## Requirements

- Skyrim Special Edition.
- BJORN – Custom Voiced Follower with Quests installed.
